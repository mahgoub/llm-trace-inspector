# v0.3.0 Release Notes

LLM Trace Inspector now has a stronger CI and integration story:

- Validate trace files against `schema/trace.schema.json`.
- Configure thresholds in `llm-trace-inspector.toml`.
- Use the project as a reusable GitHub Action.
- Read richer HTML reports with claim-level support and chunk-level ranking details.
